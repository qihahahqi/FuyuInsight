# 隐私政策

**最后更新日期：2026年4月17日**

---

## 1. 数据收集范围

本系统仅在本地收集和存储以下数据：

| 数据类型 | 用途 | 存储位置 |
|---------|------|---------|
| 用户账号信息 | 登录认证 | 本地 MySQL 数据库 |
| 持仓记录 | 投资组合管理 | 本地 MySQL 数据库 |
| 交易记录 | 历史交易追踪 | 本地 MySQL 数据库 |
| AI 分析历史 | 分析结果保存 | 本地 MySQL 数据库 |
| 回测历史 | 回测结果保存 | 本地 MySQL 数据库 |

**我们不收集、不上传、不共享任何用户数据到第三方服务器。**

---

## 2. 数据存储方式

- **本地存储**：所有数据存储在您本地 MySQL 数据库中
- **AI 调用**：AI 分析功能调用第三方 API（OpenAI/Anthropic），仅发送分析请求，不存储您的数据在第三方
- **价格数据**：从公开数据源获取市场价格数据

---

## 3. 用户数据控制权

您对您的数据拥有完全控制权：

- **查看**：可在系统中查看所有存储的数据
- **修改**：可随时修改持仓、交易等记录
- **删除**：可删除单条记录或清空所有数据
- **导出**：可导出数据为 Excel/CSV 格式

---

## 4. 数据安全措施

我们采取以下安全措施：

- **密码哈希**：用户密码使用 PBKDF2-HMAC-SHA256 哈希存储
- **JWT 认证**：使用 Token 认证，有过期时间
- **数据隔离**：多账户数据相互隔离
- **审计日志**：记录关键操作日志

---

## 5. 不收集的个人信息

我们**不收集**：

- 真实姓名、身份证号
- 手机号、邮箱（仅用于登录标识）
- 银行账户信息
- 地理位置信息
- 设备信息（超出日志记录范围）

---

## 6. 第三方服务

本系统可能使用以下第三方服务：

| 服务 | 用途 | 数据传输 |
|-----|------|---------|
| OpenAI API | AI 分析 | 仅发送分析请求文本 |
| AKShare | 价格数据获取 | 无用户数据传输 |
| Tushare | 价格数据获取 | 无用户数据传输 |

使用这些服务需要您自行配置 API Key，密钥存储在本地配置文件中。

---

## 7. 数据保留期限

- 用户数据：直到您主动删除
- 日志文件：按配置保留，默认7天
- 已删除数据：无法恢复（建议定期备份）

---

## 8. 儿童隐私

本系统不面向18岁以下未成年人。如果您是未成年人，请勿使用本系统。
