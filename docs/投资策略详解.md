## 1. 双均线策略

### 1.1 理论基础

双均线策略是最经典的技术分析策略之一，基于**趋势跟踪**原理。其核心思想是：

- **短期均线**反映近期价格趋势，对价格变化敏感
- **长期均线**反映长期价格趋势，更加平滑稳定
- 当短期均线上穿长期均线时，表明短期趋势转强，是买入信号（金叉）
- 当短期均线下穿长期均线时，表明短期趋势转弱，是卖出信号（死叉）

### 1.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| short_window | 5 | 短期均线周期（日） |
| long_window | 20 | 长期均线周期（日） |
| initial_position | 0.3 | 初始仓位比例（30%） |

### 1.3 操作逻辑

```
1. 计算短期均线 MA_short = 过去N天收盘价的平均值
2. 计算长期均线 MA_long = 过去M天收盘价的平均值
3. 当 MA_short > MA_long 时，处于多头状态
4. 当 MA_short < MA_long 时，处于空头状态
5. 当状态从空头转为多头时（金叉），买入
6. 当状态从多头转为空头时（死叉），卖出
```

### 1.4 具体操作方法

**买入条件：**
- 短期均线从下方向上穿越长期均线（金叉）
- 当前无持仓
- 按初始仓位比例（默认30%）买入

**卖出条件：**
- 短期均线从上方向下穿越长期均线（死叉）
- 当前有持仓
- 清仓卖出

**优点：** 简单易懂，趋势明确时效果好

**缺点：** 震荡市容易频繁交易，产生较多假信号

## 2. 布林带策略

### 2.1 理论基础

布林带（Bollinger Bands）由约翰·布林格发明，基于**统计学的标准差**原理：

- 价格在大部分时间内会在一定范围内波动
- 布林带由中轨（均线）、上轨、下轨组成
- 上轨 = 中轨 + K×标准差
- 下轨 = 中轨 - K×标准差
- 价格触及下轨表示可能超卖，触及上轨表示可能超买

### 2.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| window | 20 | 计算周期（日） |
| k | 2.0 | 标准差倍数 |
| initial_position | 0.3 | 初始仓位比例 |

### 2.3 操作逻辑

```
1. 计算20日移动平均线 MA（中轨）
2. 计算20日价格标准差 Std
3. 计算上轨 Upper = MA + 2 × Std
4. 计算下轨 Lower = MA - 2 × Std
5. 当价格跌破下轨时，买入（超卖反弹）
6. 当价格突破上轨时，卖出（超买回落）
```

### 2.4 具体操作方法

**买入条件：**
- 价格从上方跌破下轨（超卖区域）
- 当前无持仓
- 等待价格回到下轨上方时买入

**卖出条件：**
- 价格从下方突破上轨（超买区域）
- 当前有持仓
- 清仓卖出

**优点：** 能够捕捉价格的超买超卖区域，适合震荡市

**缺点：** 在强趋势行情中可能错过大利润

## 3. RSI策略

### 3.1 理论基础

RSI（Relative Strength Index，相对强弱指标）由韦尔德发明，用于衡量价格变动的速度和变化：

- RSI = 100 - 100/(1+RS)，其中 RS = 平均上涨幅度/平均下跌幅度
- RSI 值在 0-100 之间
- RSI < 30 为超卖区域，RSI > 70 为超买区域
- 利用超买超卖区域进行反向交易

### 3.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| period | 14 | RSI计算周期 |
| oversold | 30 | 超卖阈值 |
| overbought | 70 | 超买阈值 |
| initial_position | 0.3 | 初始仓位比例 |

### 3.3 操作逻辑

```
1. 计算每日价格变动
2. 分离上涨变动和下跌变动
3. 计算N日平均上涨幅度和平均下跌幅度
4. 计算 RS = 平均上涨/平均下跌
5. 计算 RSI = 100 - 100/(1+RS)
6. 当 RSI 从超卖区域（<30）向上突破30时，买入
7. 当 RSI 从超买区域（>70）向下跌破70时，卖出
```

### 3.4 具体操作方法

**买入条件：**
- RSI 从超卖区域（<30）向上突破30
- 表明下跌动能减弱，可能反弹
- 当前无持仓时买入

**卖出条件：**
- RSI 从超买区域（>70）向下跌破70
- 表明上涨动能减弱，可能回调
- 当前有持仓时卖出

**优点：** 能够提前预警价格反转，适合捕捉转折点

**缺点：** 在强趋势中可能过早卖出

## 4. 动量策略

### 4.1 理论基础

动量策略基于**动量效应**：过去表现好的资产在未来一段时间内往往继续表现好。核心思想是"追涨杀跌"：

- 计算一段时间内的收益率（动量）
- 当动量为正且超过阈值时，说明处于上涨趋势，买入
- 当动量为负且低于阈值时，说明处于下跌趋势，卖出

### 4.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| window | 20 | 动量计算周期（日） |
| buy_threshold | 0.05 | 买入阈值（5%） |
| sell_threshold | -0.05 | 卖出阈值（-5%） |
| initial_position | 0.3 | 初始仓位比例 |

### 4.3 操作逻辑

```
1. 计算N日动量 = 当前价格 / N日前价格 - 1
2. 当动量 > 买入阈值（5%）时，认为上涨趋势确立
3. 当动量 < 卖出阈值（-5%）时，认为下跌趋势确立
4. 动量从负转正并超过阈值时买入
5. 动量从正转负并低于阈值时卖出
```

### 4.4 具体操作方法

**买入条件：**
- N日收益率超过5%
- 说明近期有较强上涨动量
- 当前无持仓时买入

**卖出条件：**
- N日收益率低于-5%
- 说明近期有较强下跌动量
- 当前有持仓时卖出

**优点：** 能捕捉中期趋势，操作简单

**缺点：** 信号滞后，可能错过最佳买卖点

## 5. 网格交易策略

### 5.1 理论基础

网格交易策略是一种**区间震荡策略**，核心思想是：

- 在一定价格区间内设置多个网格
- 价格下跌时分批买入，价格上涨时分批卖出
- 不预测市场方向，通过低买高卖赚取差价
- 特别适合震荡行情

### 5.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| grid_count | 10 | 网格数量 |
| grid_range | 0.20 | 网格范围（上下各20%） |
| base_position | 0.5 | 初始仓位 |
| initial_position | 0.5 | 初始仓位比例 |

### 5.3 操作逻辑

```
1. 确定基准价格（首次买入价格）
2. 计算网格区间：
   - 上界 = 基准价格 × (1 + 20%)
   - 下界 = 基准价格 × (1 - 20%)
3. 将区间划分为N个网格
4. 每个网格对应一定仓位
5. 当价格下跌到下一个网格时，买入
6. 当价格上涨到上一个网格时，卖出
```

### 5.4 具体操作方法

**初始设置：**
- 以首日收盘价为基准价格
- 设置网格区间（默认上下各20%）
- 按初始仓位比例建底仓

**买入条件：**
- 价格下跌，进入更低的网格
- 按网格大小买入对应数量

**卖出条件：**
- 价格上涨，进入更高的网格
- 按网格大小卖出对应数量

**优点：** 震荡市收益稳定，不需要判断趋势

**缺点：** 单边行情会造成较大亏损或踏空

## 6. 金字塔加仓策略 ⭐ 表现最佳

### 6.1 理论基础

金字塔加仓策略是一种**趋势跟踪+仓位管理**相结合的策略：

- **核心思想**：在盈利时逐步减仓锁定利润，在亏损时逐步加仓降低成本
- **名称来源**：建仓初期仓位最大，随着价格上涨逐步减仓，仓位呈现金字塔形状
- **风险控制**：通过分批止盈锁定收益，通过分批加仓摊低成本

### 6.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| initial_position | 0.6 | 初始仓位比例（60%） |
| stop_profit_levels | [0.15, 0.18, 0.20] | 止盈触发点（盈利15%、18%、20%） |
| stop_profit_ratios | [0.30, 0.30, 0.40] | 止盈卖出比例（30%、30%、40%） |
| add_position_levels | 见下表 | 加仓触发点及比例 |
| max_add_ratio | 1.0 | 最大加仓比例 |

**加仓触发点：**

| 浮亏比例 | 加仓比例 |
|----------|----------|
| -5% | 加仓5% |
| -10% | 加仓10% |
| -15% | 加仓15% |
| -20% | 加仓20% |
| -25% | 加仓25% |

### 6.3 操作逻辑

```
阶段一：建仓
├── 首日按初始仓位比例（60%）买入
└── 剩余40%资金用于加仓

阶段二：持仓管理
├── 盈利时（价格 > 成本价）
│   ├── 盈利达到15% → 卖出30%仓位（止盈）
│   ├── 盈利达到18% → 卖出30%仓位（止盈）
│   └── 盈利达到20% → 卖出40%仓位（止盈清仓）
│
└── 亏损时（价格 < 成本价）
    ├── 浮亏5% → 加仓5%
    ├── 浮亏10% → 加仓10%
    ├── 浮亏15% → 加仓15%
    ├── 浮亏20% → 加仓20%
    └── 浮亏25% → 加仓25%（最大）
```

### 6.4 具体操作方法

**第一步：建仓**
- 选择有投资价值的标的
- 首日按60%仓位买入
- 保留40%现金用于加仓

**第二步：止盈操作（盈利时）**

| 触发条件 | 操作 | 剩余仓位 |
|----------|------|----------|
| 盈利15% | 卖出30% | 剩70% |
| 盈利18% | 再卖出30% | 剩40% |
| 盈利20% | 清仓卖出 | 0% |

**第三步：加仓操作（亏损时）**

| 浮亏 | 加仓比例 | 说明 |
|------|----------|------|
| -5% | 加仓5% | 小幅补仓 |
| -10% | 加仓10% | 中度补仓 |
| -15% | 加仓15% | 较大补仓 |
| -20% | 加仓20% | 大幅补仓 |
| -25% | 加仓25% | 最大补仓 |

### 6.5 策略优势

1. **风险可控**：分批止盈确保盈利不会回吐
2. **成本优化**：下跌时加仓降低持仓成本
3. **情绪稳定**：有明确的操作规则，避免情绪化决策
4. **适合长期**：适合优质股票或基金的长期持有

### 6.6 策略示例

假设初始资金10万元，买入某股票：

**建仓阶段：**
- 买入价：10元/股
- 初始仓位60%：6万元买入6000股
- 剩余现金：4万元

**情景一：价格上涨**
- 价格涨到11.5元（盈利15%）
- 触发第一次止盈，卖出30%即1800股，获得20700元
- 剩余持仓：4200股，成本仍为10元

- 价格涨到11.8元（盈利18%）
- 触发第二次止盈，卖出30%即1260股，获得14868元
- 剩余持仓：2940股

- 价格涨到12元（盈利20%）
- 触发第三次止盈，清仓卖出2940股，获得35280元
- 总收益：70848元（70.8%收益）

**情景二：价格下跌**
- 价格跌到9.5元（浮亏5%）
- 触发加仓，用初始持仓市值6万的5%即3000元加仓
- 买入316股，新成本 = (6000×10 + 316×9.5) / 6316 ≈ 9.97元

- 价格跌到9元（浮亏约10%）
- 触发加仓，用初始持仓市值6万的10%即6000元加仓
- 买入667股，成本进一步降低...

### 6.7 注意事项

1. **标的选择**：只适用于优质标的，垃圾股越补越亏
2. **资金管理**：确保有足够现金用于加仓
3. **止损底线**：设置最大亏损线（如-30%），超过则止损
4. **心态控制**：严格执行纪律，不因短期波动改变计划

## 推荐使用

根据回测结果，**金字塔加仓策略**在多种市场环境下表现最为稳定，特别适合：

- 长期投资优质股票或基金
- 希望降低持仓成本的投资者
- 追求稳健收益而非暴利的投资者
- 有一定资金储备可用于加仓的投资者

**使用建议：**
1. 选择基本面良好的标的
2. 严格执行止盈加仓纪律
3. 预留足够的加仓资金
4. 设置止损底线控制风险

## 高级参数调整指南

本章节详细说明各策略的高级参数调整方法、调整范围、以及对回测结果的影响。

### 通用参数说明

所有策略都包含一个通用参数：

| 参数 | 默认值 | 调整范围 | 影响 |
|------|--------|----------|------|
| `initial_position` | 0.3（30%）| 0.1 - 1.0 | 初始建仓比例，影响资金使用率和风险敞口 |

**调整建议：**
- **保守型**：设置 0.2-0.3，保留更多现金应对加仓
- **激进型**：设置 0.5-0.7，追求更高收益但风险更大
- **全仓策略**：设置 1.0，无预留加仓资金

---

### 1. 双均线策略参数详解

#### 1.1 短期均线周期（short_window）

| 属性 | 值 |
|------|-----|
| 默认值 | 5 |
| 调整范围 | 3 - 30 |
| 建议值 | 5（快线）、10（中线）、20（慢线） |

**影响分析：**

| 设置值 | 交易频率 | 信号灵敏度 | 适用行情 |
|--------|----------|------------|----------|
| 3-5 | 高 | 极高，快速响应 | 短期波动行情 |
| 10-15 | 中 | 中等，较稳定 | 中期趋势行情 |
| 20-30 | 低 | 低，信号滞后 | 长期趋势行情 |

**调整示例：**
- 调低（如3）：适合短线交易，但会增加假信号
- 调高（如10）：减少假信号，但可能错过最佳入场点

#### 1.2 长期均线周期（long_window）

| 属性 | 值 |
|------|-----|
| 默认值 | 20 |
| 调整范围 | 20 - 250 |
| 建议值 | 20（月线）、60（季线）、120（半年线）、250（年线） |

**影响分析：**

| 设置值 | 趋势判断 | 信号稳定性 | 滞后性 |
|--------|----------|------------|--------|
| 20-30 | 短期趋势 | 较低，易受干扰 | 低 |
| 60-120 | 中期趋势 | 较高，趋势明确 | 中 |
| 120-250 | 长期趋势 | 高，大趋势判断 | 高 |

**参数组合建议：**

| 短期/长期组合 | 策略风格 | 适用场景 |
|---------------|----------|----------|
| 5/20 | 短线策略 | 日内波动较大的标的 |
| 10/60 | 中线策略 | 波段操作，2-3个月周期 |
| 20/120 | 长线策略 | 趋势投资，半年以上 |
| 60/250 | 趋势策略 | 大趋势判断，年化操作 |

### 2. 布林带策略参数详解

#### 2.1 计算周期（window）

| 属性 | 值 |
|------|-----|
| 默认值 | 20 |
| 调整范围 | 10 - 60 |
| 建议值 | 20（标准）、40（放宽） |

**影响分析：**

| 设置值 | 带宽宽度 | 交易频率 | 假信号率 |
|--------|----------|----------|----------|
| 10-15 | 较窄 | 高 | 高 |
| 20（默认）| 标准 | 中 | 中 |
| 30-60 | 较宽 | 低 | 低 |

**调整建议：**
- **震荡行情**：调低周期（10-15），捕捉更小区间波动
- **趋势行情**：调高周期（30-40），避免频繁交易
- **高波动标的**：调高周期，减少假突破信号

#### 2.2 标准差倍数（k）

| 属性 | 值 |
|------|-----|
| 默认值 | 2.0 |
| 调整范围 | 1.0 - 3.0 |
| 建议值 | 1.5（紧）、2.0（标准）、2.5（松） |

**影响分析：**

| 设置值 | 带宽 | 触发频率 | 信号可靠性 |
|--------|------|----------|------------|
| 1.5 | 窄带 | 高（频繁触发）| 较低，易假突破 |
| 2.0 | 标准 | 中（正常）| 中等 |
| 2.5 | 宽带 | 低（较少触发）| 较高，强信号 |

**调整示例：**

**调整建议：**

### 6. 金字塔加仓策略参数详解

#### 6.1 初始仓位（initial_position）

| 默认值 | 0.6（60%）|

| 0.6（默认）| 中 | 中（+40%）| 中等 |
| 0.7-0.8 | 高 | 小（+20-30%）| 高，加仓空间有限 |

**调整建议：**
- **新手投资者**：设置 0.3-0.4，预留充足加仓资金
- **有经验投资者**：设置 0.5-0.6，平衡收益与风险
- **激进投资者**：设置 0.7，追求高收益但需严格止损

#### 6.2 止盈触发点（stop_profit_levels）

| 属性 | 值 |

| 格式 | 浮点数数组，表示盈利百分比 |

**影响分析：**

| 设置方式 | 止盈节奏 | 收益锁定 | 剩余上涨空间 |
|----------|----------|----------|--------------|
| 紧凑型：[0.10, 0.15, 0.20] | 快速止盈 | 早期锁定利润 | 少，可能错过大涨 |
| 标准：[0.15, 0.18, 0.20] | 稳健止盈 | 平衡 | 中等 |
| 宽松型：[0.20, 0.30, 0.40] | 慢速止盈 | 追求高收益 | 多，但回撤风险大 |

**调整示例：**
```
# 短期策略（追求快速止盈）
stop_profit_levels: [0.08, 0.12, 0.15]

stop_profit_levels: [0.15, 0.18, 0.20]

# 长期策略（追求高收益）
```

| 属性 | 值 |

| 格式 | 浮点数数组，总和应为1.0 |

**影响分析：**

| 设置方式 | 止盈风格 | 仓位变化 |
|----------|----------|----------|
| 均衡型：[0.33, 0.33, 0.34] | 平均分批 | 每次卖约1/3 |
| 前重型：[0.50, 0.30, 0.20] | 早期多卖 | 第一次卖一半，后续递减 |
| 后重型：[0.20, 0.30, 0.50] | 晚期多卖 | 前期少卖，最后清仓 |

**调整建议：**
- **保守型**：前期多卖（如 [0.40, 0.35, 0.25]），快速锁定利润
- **激进型**：后期多卖（如 [0.20, 0.30, 0.50]），追求更高收益

#### 6.4 加仓触发点（add_position_levels）

| 属性 | 值 |

| -10% | 中度亏损 | 10% | 15% |
| -15% | 较大亏损 | 15% | 30% |
| -20% | 大幅亏损 | 20% | 50% |
| -25% | 严重亏损 | 25% | 75%（接近上限）|

**调整策略：**

| 策略类型 | 加仓设置 | 特点 |
|----------|----------|------|
| 保守型 | 最大加仓到-15% | 控制风险，早止损 |
| 标准（默认）| 最大加仓到-25% | 平衡摊成本与风险 |
| 激进型 | 最大加仓到-35% | 深度摊成本，需充足资金 |

**自定义示例：**
